Privacy Policy

Last updated: April 2026

This application is an internal tool used solely by its developer to publish video content to owned TikTok accounts. It is not a public-facing product.

1. Who We Are

This application ("App_management") is developed and operated by Yannick Leggeri ("I", "me", or "my") for the sole purpose of managing and publishing video content to TikTok accounts under my ownership.

2. What Data This App Accesses

To function, the App requests the following access through TikTok's official API:

  • Basic TikTok account information (username, profile ID)
  • Permission to upload and publish video content to owned TikTok accounts

No data is collected from any third-party TikTok users.

3. How Data Is Used

All data accessed through TikTok's API is used exclusively to:

  • Authenticate with TikTok's Content Posting API
  • Upload and publish video content to accounts I own
  • Manage OAuth tokens required for API access

Data is never sold, shared, or disclosed to any third parties.

4. Data Storage

OAuth access tokens and refresh tokens are stored locally and securely on my own systems. These tokens are used only to authenticate API requests and are not shared with any external service.

5. Third-Party Services

This App interacts exclusively with TikTok's official API. All interactions are governed by TikTok's Privacy Policy.

6. Data Retention

API tokens are retained only as long as necessary to operate the App. They can be revoked at any time by disconnecting the App from your TikTok account settings.

7. Your Rights

Since this App only accesses accounts I personally own, no third-party data subject rights apply. You may revoke App access at any time via TikTok's connected apps settings.
